# Envipe-2021-2025
Proyecto dedicado a analizar 10 preguntas sobre la Encuesta Nacional de Victimización y Percepción sobre Seguridad Pública desde el año 2021 hasta 2025
Para ejecutar el código debe encontrarse en un compilador de Rstudio, tener los archivos adjuntos descargados en un mismo set directionary working, posteriormente debe descargar las librerias indicadas en el código y ejecutarlo como R mark down.
